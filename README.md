2015年度 データベース設計論
====

## 学習目標
* リレーショナルデータベースを用いたアプリケーションを設計・実装するための一通りの知識を習得する

## 授業形式
* 講義＋演習
* 半年かけてデータベース設計の演習をします
 * ３人までのグループを作ってグループワークをします
 * 作成したいデータベースを各自考えておくこと
* 成績は試験とレポート、出席（小テスト）の合計
